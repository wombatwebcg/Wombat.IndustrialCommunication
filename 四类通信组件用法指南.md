# 六类通信组件用法指南（跨项目接入）

## 1. 适用范围

本文面向外部项目，说明以下六个组件的定位、生命周期、读写方式与接入注意事项：

- `ModbusRtuClient`
- `ModbusTcpClient`
- `SiemensClient`
- `ModbusRtuServer`
- `ModbusTcpServer`
- `S7TcpServer`

---

## 2. 快速结论

- `ModbusRtuClient`、`ModbusTcpClient`、`SiemensClient` 都是客户端，走 `Connect()/Disconnect()` 生命周期。
- `ModbusRtuServer`、`ModbusTcpServer` 是从站服务器，核心是维护 `DataStore`，其 `Read/Write` 类接口为保护性不支持实现。
- `S7TcpServer` 是可模拟西门子 PLC 的服务器，既可协议响应，也支持本地按区域或按地址字符串读写。
- Modbus 客户端均支持标准地址 `站号;功能码;地址` 和增强地址 `站号;逻辑地址`（如 `1;40001`）。
- 客户端均支持长连接/短连接模式；长连接模式下可结合自动重连。

---

## 3. ModbusRtuClient 用法

### 3.1 构造与连接

```csharp
using Wombat.IndustrialCommunication.Modbus;
using System.IO.Ports;

var rtu = new ModbusRtuClient(
    portName: "COM3",
    baudRate: 9600,
    dataBits: 8,
    stopBits: StopBits.One,
    parity: Parity.None,
    handshake: Handshake.None);

rtu.IsLongConnection = true;
rtu.EnableAutoReconnect = true;
rtu.MaxReconnectAttempts = 5;
rtu.ReconnectDelay = TimeSpan.FromSeconds(2);

var connect = rtu.Connect();
if (!connect.IsSuccess) throw new Exception(connect.Message);
```

### 3.2 同步读写（IModbusClient 快捷接口）

```csharp
byte station = 1;

var w1 = rtu.WriteCoil(station, 0, true);
var r1 = rtu.ReadCoil(station, 0);

var w2 = rtu.WriteHoldingRegister(station, 0, 1234);
var r2 = rtu.ReadHoldingRegister(station, 0);
```

### 3.3 异步与批量读写

```csharp
using Wombat.Extensions.DataTypeExtensions;

await rtu.ConnectAsync();
await rtu.WriteCoilAsync(1, 0, true);
var coil = await rtu.ReadCoilAsync(1, 0);

var writeMap = new Dictionary<string, (DataTypeEnums, object)>
{
    { "1;5;0", (DataTypeEnums.Bool, true) },
    { "1;6;1", (DataTypeEnums.UInt16, (ushort)100) }
};
await rtu.BatchWriteAsync(writeMap);

var readMap = new Dictionary<string, DataTypeEnums>
{
    { "1;1;0", DataTypeEnums.Bool },
    { "1;3;1", DataTypeEnums.UInt16 }
};
var batch = await rtu.BatchReadAsync(readMap);
```

## 4. ModbusTcpClient 用法

### 4.1 构造与连接

```csharp
using Wombat.IndustrialCommunication.Modbus;

var tcp = new ModbusTcpClient("192.168.1.10", 502);
tcp.IsLongConnection = true;
tcp.EnableAutoReconnect = true;
tcp.MaxReconnectAttempts = 5;
tcp.ReconnectDelay = TimeSpan.FromSeconds(2);

var connect = tcp.Connect();
if (!connect.IsSuccess) throw new Exception(connect.Message);
```

### 4.2 同步读写（与 RTU 客户端一致）

```csharp
var w1 = tcp.WriteCoil(1, 0, true);
var r1 = tcp.ReadCoil(1, 0);

var w2 = tcp.WriteHoldingRegister(1, 0, 1234);
var r2 = tcp.ReadHoldingRegister(1, 0);
```

### 4.3 异步读写

```csharp
await tcp.ConnectAsync();
await tcp.WriteHoldingRegisterAsync(1, 0, 1234);
var reg = await tcp.ReadHoldingRegisterAsync(1, 0);
await tcp.DisconnectAsync();
```

## 5. SiemensClient 用法

### 5.1 构造与连接

```csharp
using Wombat.IndustrialCommunication.PLC;

var plc = new SiemensClient(
    ip: "192.168.1.20",
    port: 102,
    siemensVersion: SiemensVersion.S7_1200,
    slot: 1,
    rack: 0);

plc.IsLongConnection = true;
plc.EnableAutoReconnect = true;
plc.MaxReconnectAttempts = 3;
plc.ReconnectDelay = TimeSpan.FromSeconds(5);

var connect = await plc.ConnectAsync();
if (!connect.IsSuccess) throw new Exception(connect.Message);
```

### 5.2 地址字符串读写

```csharp
plc.Write("DB1.DBW0", (ushort)32000);
var w = plc.ReadUInt16("DB1.DBW0");

plc.Write("DB1.DBX2.0", true);
var x = plc.ReadBoolean("DB1.DBX2.0");

plc.Write("M10", (byte)0x12);
var m = plc.ReadByte("M10");
```

### 5.3 注意事项

- `SiemensClient` 构造时直接确定 `SiemensVersion`、`slot`、`rack`，不通过 `SetRackSlot` 设置。
- 读写能力来自其 S7 通信基类，建议按 `DB/M/I/Q/V` 地址规范统一建模。

## 6. ModbusRtuServer 用法

### 6.1 构造与监听

```csharp
using Wombat.IndustrialCommunication.Modbus;
using System.IO.Ports;

var rtuServer = new ModbusRtuServer("COM3", 9600, 8, StopBits.One, Parity.None, Handshake.None);
var listen = rtuServer.Listen();
if (!listen.IsSuccess) throw new Exception(listen.Message);
```

### 6.2 核心操作：维护 DataStore

```csharp
rtuServer.DataStore.CoilDiscretes[0] = true;
rtuServer.DataStore.InputDiscretes[0] = false;
rtuServer.DataStore.HoldingRegisters[0] = 1234;
rtuServer.DataStore.InputRegisters[0] = 5678;
```

### 6.3 重要注意

- 类上大量 `ReadXxx/WriteXxx/BatchXxx` 接口会返回不支持，这是设计行为。
- 服务器正确用法是维护 `DataStore`，由外部 Modbus 客户端发起协议读写。

---

## 7. ModbusTcpServer 用法

### 7.1 构造与监听

```csharp
using Wombat.IndustrialCommunication.Modbus;

var tcpServer = new ModbusTcpServer("0.0.0.0", 502);
tcpServer.MaxConnections = 200;
var listen = tcpServer.Listen();
if (!listen.IsSuccess) throw new Exception(listen.Message);
```

### 7.2 维护 DataStore

```csharp
tcpServer.DataStore.CoilDiscretes[0] = true;
tcpServer.DataStore.HoldingRegisters[0] = 1000;
```

### 7.3 重要注意

- `Read/Write/BatchRead/BatchWrite` 接口均为不支持返回。
- 业务应将其视为协议网关与内存镜像，不应把它当作主动采集客户端。

## 8. S7TcpServer 用法

### 8.1 构造、PLC 参数、监听

```csharp
using Wombat.IndustrialCommunication.PLC;

var s7Server = new S7TcpServer("0.0.0.0", 102);
s7Server.SetSiemensVersion(SiemensVersion.S7_1200);
s7Server.SetRackSlot(0, 1);
s7Server.CreateDataBlock(1, 1024);

var listen = s7Server.Listen();
if (!listen.IsSuccess) throw new Exception(listen.Message);
```

### 8.2 按区域直接操作

```csharp
s7Server.WriteDB(1, 0, new byte[] { 0x00, 0x64 });
var db = s7Server.ReadDB(1, 0, 2);

s7Server.WriteMerkers(0, new byte[] { 1, 2, 3, 4 });
var m = s7Server.ReadMerkers(0, 4);
```

### 8.3 按地址字符串读写（IReadWrite 风格）

```csharp
using Wombat.Extensions.DataTypeExtensions;

s7Server.Write("DB1.DBW0", (ushort)32000);
var w = s7Server.ReadUInt16("DB1.DBW0");

s7Server.Write("DB1.DBX2.0", true);
var x = s7Server.ReadBoolean("DB1.DBX2.0");

s7Server.Write("V100", (byte)0x34);
var v = s7Server.ReadByte("V100");
```

### 8.4 地址规则

- DB 区：`DB1.DBX0.0` / `DB1.DBB0` / `DB1.DBW0` / `DB1.DBD0`
- M 区：`M0`
- I 区：`I0`
- Q 区：`Q0`
- T 区：`T0`
- C 区：`C0`
- V 区：`V100`（内部映射到 `DB1`）

### 8.5 数据事件

```csharp
s7Server.DataWritten += (_, e) =>
{
    Console.WriteLine($"{e.Area} {e.StartAddress} {e.Data?.Length}");
};

s7Server.DataRead += (_, e) =>
{
    Console.WriteLine($"{e.Area} {e.StartAddress} {e.Data?.Length}");
};
```

## 9. Modbus 地址格式说明（RTU/TCP 客户端通用）

- 标准格式：`站号;功能码;地址`，例如 `1;3;0`
- 增强格式：`站号;逻辑地址`，例如：
  - `1;40001` 保持寄存器，映射到协议地址 `0`
  - `1;30001` 输入寄存器
  - `1;20001` 离散输入
  - `1;10001` 线圈

---

## 10. 生命周期建议

- `ModbusRtuClient`：`Connect -> Read/Write -> Disconnect`
- `ModbusTcpClient`：`Connect -> Read/Write -> Disconnect`
- `SiemensClient`：`Connect -> Read/Write -> Disconnect`
- `ModbusRtuServer`：`Listen -> 维护 DataStore -> Shutdown`
- `ModbusTcpServer`：`Listen -> 维护 DataStore -> Shutdown`
- `S7TcpServer`：`CreateDataBlock/初始化区域 -> Listen -> Shutdown`

---

## 11. 跨项目接入最小模板

```csharp
using Wombat.IndustrialCommunication.Modbus;
using Wombat.IndustrialCommunication.PLC;
using Wombat.Extensions.DataTypeExtensions;
```

- Modbus 主站采集：`ModbusRtuClient` 或 `ModbusTcpClient`
- Modbus 从站模拟：`ModbusRtuServer` 或 `ModbusTcpServer`，并维护 `DataStore`
- 西门子设备访问：`SiemensClient`
- 西门子 PLC 模拟：`S7TcpServer`

---

## 12. 停止与释放

```csharp
rtu?.Disconnect();
tcp?.Disconnect();
plc?.Disconnect();
rtuServer?.Shutdown();
tcpServer?.Shutdown();
s7Server?.Shutdown();
```

建议在宿主进程退出时统一执行停止，避免端口或串口资源残留占用。
